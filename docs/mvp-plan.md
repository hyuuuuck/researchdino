# MVP Plan

## Phase 1: Repository Foundation

- Create initial repo.
- Capture product vision and architecture.
- Choose frontend and backend scaffold.

## Phase 2: Visual Prototype

- Build laboratory map.
- Add room detail panels.
- Add mock paper cards moving through the workflow.
- Add status events and basic timeline.

## Phase 3: Local Paper Ingestion

- Register a local PDF folder.
- Extract PDF metadata and text.
- Store parsed papers locally.
- Show paper cards in Collection Dock and Reading Bench.

## Phase 4: Agent Workflow

- Implement Reader output schema.
- Implement Debate transcript schema.
- Implement Leader approval queue.
- Store approved knowledge in Library.

## Phase 5: Research Outputs

- Generate hypothesis cards.
- Generate experiment design cards.
- Generate manuscript outlines in Writing Studio.
